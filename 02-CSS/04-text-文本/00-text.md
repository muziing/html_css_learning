# CSS 文本属性

CSS Text（文本）属性可以定义文本的**外观**，比如文本的颜色、对齐文本、装饰文本、文本缩进、行间距等

## 文本颜色

**color**属性用于定义文本的颜色

```css
div {
    color: red;
}
```



|  表示   |  属性值  |
|--------|----------|
| 预定义的颜色值 | red, green, blue, ... |
| 十六进制 | #FF0000, #FF6600, #29D794 |
| RGB代码 | rgb(255, 0, 0) 或 rgb(100%, 0%, 0%) |

**开发中最常用的是十六进制**
